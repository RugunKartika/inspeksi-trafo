# inspeksi-trafo
⚡ Inspeksi Trafo

Aplikasi web berbasis Laravel untuk sistem inspeksi trafo yang digunakan untuk pencatatan, pengelolaan, dan monitoring data inspeksi secara digital.

📌 Tentang Project

Project ini dibuat untuk membantu proses inspeksi trafo agar lebih mudah, cepat, dan terstruktur. Semua data inspeksi dapat disimpan dan dikelola melalui sistem berbasis web.
